# PortManager 界面设计说明

适用版本：v1.1.0

## 设计目标

- 信息优先：先看到端口状态，再做操作
- 决策友好：关闭前提供可理解的端口建议
- 干扰最小：扫描和系统命令执行时不弹命令窗口
- 稳定优先：窗口在最小化恢复后仍保持预期位置与尺寸

## 布局结构

界面采用单窗口卡片布局（固定尺寸）：

1. 扫描卡片：快速扫描、全面扫描、关闭端口、状态提示
2. 结果卡片：端口列表、计数、右键洞察入口
3. 设置卡片：开机自启动、关于、退出、刷新

## 视觉规范（当前实现）

- 主背景：#F5F4ED
- 卡片背景：#FAF9F5
- 主文本：#141413
- 次文本：#5E5D59 / #4D4C48
- 默认中文字体优先：Microsoft YaHei UI
- 标题字体优先：Anthropic Serif，回退 Georgia / Times New Roman

## 交互规范

### 扫描

- 同一时刻仅允许一个扫描任务
- 重复点击会提示扫描进行中
- 完成后展示端口数量与耗时

### 关闭端口

- 仅对当前选中条目生效
- 关闭后自动触发一次全面扫描

### 右键洞察

- 对选中端口展示用途、连接活跃度和建议动作
- 建议关闭时使用警告图标

### 开机自启动

- 勾选即写入注册表 Run 项
- 失败时自动回滚勾选状态并提示错误

## 可用性约束

- 工具定位为本机端口管理，不提供远程能力
- 关闭端口属于高影响动作，需用户主动选择并确认
- 对系统核心进程占用端口默认倾向建议开启

## 后续可迭代方向

- 增加结果过滤与搜索
- 增加导出扫描结果
- 增加关键端口保护名单
