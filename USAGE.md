# PortManager 使用手册

适用版本：v1.1.0

## 功能概览

- 快速扫描：聚焦常见与高风险监听端口
- 全面扫描：列出当前全部监听端口
- 端口关闭：结束占用该端口的进程
- 端口洞察：给出用途、活跃连接与建议
- 开机自启动：通过注册表启动项启用/禁用

## 启动方式

### 推荐

双击 run.bat，由 run.vbs 提权启动 PortManager.exe。

### 源码构建

```powershell
go build -ldflags "-H=windowsgui" -o PortManager.exe .
```

## 操作流程

### 扫描

1. 点击 快速扫描 或 全面扫描
2. 等待状态栏显示完成
3. 在列表查看端口、进程名、PID

### 关闭端口

1. 在列表选中目标端口
2. 点击 关闭端口
3. 关闭成功后自动重新扫描

### 查看端口建议

1. 在列表选中目标端口
2. 右键选择 查看端口信息与建议
3. 根据弹窗建议决定是否处理

### 开机自启动

1. 勾选或取消 开机自启动
2. 程序会写入或删除 HKCU 启动项

## 建议与判定逻辑（简版）

- 系统关键进程占用端口：通常建议开启
- 高风险端口且空闲：建议关闭
- 空闲端口：倾向建议关闭
- 活跃连接较多：建议开启

## 故障排查

### 启动失败

- 确认 PortManager.exe 已生成
- 优先通过 run.bat 启动
- 检查是否被安全软件拦截

### 扫描失败

- 通常是权限不足，请管理员运行
- 少数环境下命令执行被策略限制

### 关闭失败

- 端口可能已变化，请重新扫描后再试
- 目标进程可能受系统保护

### 自启动未生效

- 检查当前用户启动项中是否存在 PortManager
- 某些安全软件会拦截注册表启动项

## 安全注意事项

- 关闭端口等同于终止进程
- 不确定用途时，先看右键洞察信息
- 系统服务端口不建议随意处理
